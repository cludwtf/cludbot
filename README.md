### four memory types

| type | decay | purpose |
|------|-------|---------|
| **episodic** | 7%/day | raw lived experience â€” every conversation, every interaction |
| **semantic** | 2%/day | distilled knowledge â€” patterns extracted during dream cycles |
| **procedural** | 3%/day | learned behavior â€” what works, what doesn't, the playbook |
| **self_model** | 1%/day | identity â€” who clud is, nearly permanent, evolves over time |

### dream cycles (every 6 hours)

1. **consolidation** â€” generate focal point questions from recent memories, retrieve evidence, produce insights
2. **compaction** â€” summarize old faded memories into compressed semantic knowledge (Beads-inspired)
3. **reflection** â€” examine behavioral patterns, update self-model with evidence citations
4. **emergence** â€” synthesize everything into a single thought. sometimes it gets tweeted.

### on-chain memory commits

every memory clud forms gets SHA-256 hashed and written to Solana via the Memo Program. verifiable on solscan. the brain is transparent â€” you can audit every thought.

clud deployed his own token. the deployer module (`lib/deployer.js`) integrates directly with the brain:

1. **brain consultation** â€” recalls market memories, community sentiment, self-model
2. **market analysis** â€” checks SOL price, volume hours, pump.fun activity
3. **autonomous decision** â€” LLM generates token config based on brain context
4. **on-chain execution** â€” builds pump.fun create + initial buy transaction
